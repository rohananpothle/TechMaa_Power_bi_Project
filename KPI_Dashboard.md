# üß≠ Productivity KPIs

| KPI                           | Recommended Visual      | Description                                                       |
| ----------------------------- | ----------------------- | ----------------------------------------------------------------- |
| Active Working Hours          | Line Chart / Area Chart | Track daily or weekly working hours per employee.                 |
| Idle Time Ratio               | Gauge Chart             | Displays % of idle vs active time during work hours.              |
| Daily Work Contribution Score | Clustered Column Chart  | Compare contribution scores between employees or teams.           |
| Task Completion Percentage    | KPI Card / Gauge        | Shows completed tasks as a percentage of assigned ones.           |
| Task Efficiency               | Bar Chart               | Visualizes `(Tasks Completed / Tasks Assigned)` across employees. |


# üïí Attendance & Participation KPIs

| KPI                                        | Recommended Visual   | Description                                                   |
| ------------------------------------------ | -------------------- | ------------------------------------------------------------- |
| Daily Session Attendance Rate              | Line Chart           | Trends of attendance over days or weeks.                      |
| Engagement in Live Sessions                | Stacked Column Chart | Represents session interaction levels (Q&A, Duration, Chats). |
| Learning & Development Participation Score | Treemap              | Shows engagement across L&D sessions or individuals.          |


# üßÆ Performance KPIs:

| KPI                    | Recommended Visual   | Description                                                |

| Quality Score          | Radar / Column Chart | Compares evaluation-based quality scores per team.         |
| Performance Index      | Scatter / Line Chart | Tracks overall weighted performance index trend over time. |

# üìò 2. DAX Formulas

# üß≠ Productivity KPIs

    Active Working Hours =
    SUM('WorkLog'[ActiveHours])
    
    Idle Time Ratio (%) =
    DIVIDE(SUM('WorkLog'[IdleHours]), SUM('WorkLog'[TotalHours]), 0) * 100
    
    Daily Work Contribution Score =
    AVERAGE('WorkLog'[ContributionScore])
    
    Task Completion Percentage (%) =
    DIVIDE(SUM('Tasks'[CompletedTasks]), SUM('Tasks'[AssignedTasks]), 0) * 100
    
    Task Efficiency (%) =
    DIVIDE(SUM('Tasks'[CompletedTasks]), SUM('Tasks'[AssignedTasks]), 0) * 100

# üïí Attendance & Participation KPIs

    Daily Session Attendance Rate (%) =
    DIVIDE(SUM('Attendance'[SessionsAttended]), SUM('Attendance'[SessionsScheduled]), 0) * 100
    
    Engagement in Live Sessions =
    SUM('Attendance'[EngagementPoints])
    
    Learning & Development Participation Score =
    SUM('Learning'[TrainingPoints])


# üßÆ Performance KPIs

    Timely Submission Rate (%) =
    DIVIDE(SUM('Tasks'[CompletedOnTime]), SUM('Tasks'[TotalCompleted]), 0) * 100
    
    Quality Score =
    AVERAGE('Evaluation'[QualityScore])
    
    Performance Index =
    (0.4 * [Task Completion Percentage (%)]) +
    (0.3 * [Quality Score]) +
    (0.3 * [Daily Session Attendance Rate (%)])

# üí∞ Revenue & Conversion KPIs

    Leads Generated =
    COUNT('CRM'[LeadID])
    
    Lead-to-Sale Conversion Ratio (%) =
    DIVIDE(COUNTROWS(FILTER('CRM', 'CRM'[Status] = "Closed Won")), COUNTROWS('CRM'), 0) * 100
    
    Revenue Impact Score =
    SUM('CRM'[RevenueGenerated])
    
    Monthly Revenue Contribution =
    CALCULATE(SUM('CRM'[RevenueGenerated]), DATESMTD('Date'[Date]))


# üìä Additional Time Intelligence

    Previous Month Revenue =
    CALCULATE([Monthly Revenue Contribution], PREVIOUSMONTH('Date'[Date]))
    
    Revenue Growth (%) =
    DIVIDE([Monthly Revenue Contribution] - [Previous Month Revenue], [Previous Month Revenue], 0) * 100
    
    YTD Revenue =
    TOTALYTD(SUM('CRM'[RevenueGenerated]), 'Date'[Date])
